# 💰 Expense Tracker

A simple command-line Expense Tracker built using Python. This project allows users to enter multiple expenses and calculates the total amount spent.

## 📌 Features

- Add multiple expenses
- Automatically calculate the total
- Menu-free, beginner-friendly program
- Uses Python loops and arithmetic operations

## 🛠️ Technologies Used

- Python 3

## 📂 Project Structure

```
Expense-Tracker/
│── expense_tracker.py
└── README.md
```

## ▶️ How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open the project folder in a terminal.
4. Run:

```bash
python expense_tracker.py
```

## 📖 Example

```
Enter expense amount: ₹100
Enter expense amount: ₹50
Enter expense amount: ₹20

Total Spent: ₹170
```

## 🎯 Concepts Used

- Variables
- User Input
- While Loop
- Arithmetic Operations
- Accumulator (`total = total + expense`)
